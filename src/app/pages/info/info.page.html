Sobre a API

A API usada neste projeto é a Rest Countries API, é uma API pública que fornece informações detalhadas sobre todos os países do mundo. Ela retorna dados no formato JSON e pode ser utilizada em aplicações que envolvem geografia, turismo, ou qualquer funcionalidade que necessite de informações sobre países. Fornece dados como: nome do país, população, região, capital, entre outros.
